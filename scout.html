Artificially intelligent

The Scout AI Core is equipped a specialized, onboard AI chip. Each unit can remotely download and run customizable TensorFlow Lite models for enhanced anomoly detection, improving:

  • - Latency: real-time anomoly detection and response within milliseconds
  • - Connectivity: annotated data available even if satellite data is temporarily affected by storm clouds
  • - Flexibility: customizable anomoly detection for any application

Unlimited satellite data and reliable solar power

Continuous, round-the-clock data streaming over a dedicated satellite link. 5V/12V/24V sensors are powered by a solar panel and battery system specially designed to withstand up to three days without sun. Since each terminal is completely off-the-grid, it will continue to stream even if nearby infrastructure is disabled.

Compatible with most existing sensors and cameras

Scout AI Core works with most existing third-party sensors and cameras with Ethernet, RS-232, RS-435, Modbus, and Profibus protocol compatibility with DB9 and RJ-45 hardware interfaces.
